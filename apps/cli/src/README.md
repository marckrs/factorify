# CLI — Factorify

Interface de linha de comando para interagir com a plataforma.

## Uso esperado
  factorify run "construa um sistema de autenticação JWT"
  factorify status
  factorify logs --agent code --last 10
  factorify approve --task-id abc123

## Stack
- Node.js + TypeScript
- Commander.js para parsing de comandos
- Publicado no npm como `factorify`
