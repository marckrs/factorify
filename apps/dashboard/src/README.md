# Dashboard — Factorify

Painel de controle humano da plataforma autônoma.

## Responsabilidades
- Status em tempo real de todos os agentes
- Visualização de OKRs e progresso
- Log de execuções do orquestrador
- Aprovação de deploys em produção
- Métricas da plataforma (tasks executadas, tokens usados, tempo médio)

## Stack
- React + TypeScript + Tailwind CSS
- Supabase Realtime para atualizações ao vivo
- Vercel para deploy
